
## httpProxy

```makefile
#for server
cd CiProxy/server 
# -ip (server ip default all interface) -port (server port) -method (direct and tunnel)
./main

#for client(need enable proxy server maually)
cd CiProxy/client
# -ip (server ip) -port (server port)
./main

```
