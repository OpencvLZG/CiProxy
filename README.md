#  <center>  CiProxy </center>

###### <center>Introduce</center>

<center>use tcp implement, not http server, base on golang</center>

###### install

```makefile
#makefile for server

apt install go

git clone https://github.com/OpencvLZG/CiProxy

#for server
cd CiProxy/server 

go build main.go

#for client
cd CiProxy/client

go build main.go
```

###### running

```makefile
#for server
cd CiProxy/server 

./main

#for client(need enable proxy server maually)
cd CiProxy/client

./main

```
